# POC Middleware — Slidev

Repositório dos slides da apresentação **POC Middleware — EME4** da Datainfo, mantida em [Slidev](https://sli.dev).

> A documentação técnica do projeto (análises, planos, contratos, estudos) foi migrada em **04/05/2026** para o repositório privado [`Middleware`](https://github.com/fabriciomoeller/Middleware) — pasta `docs/`. Este repositório fica responsável apenas pelos slides.

## Como rodar a apresentação

```bash
cd poc-middleware-slidev
npm install
npm run dev
```

Os slides ficam em `poc-middleware-slidev/slides/` e são orquestrados a partir de `poc-middleware-slidev/slides.md`.
